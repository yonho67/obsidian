# 《시간, 공간, 인간》 확장본3 — 복선-회수 구조

> 각 꼭지가 어떤 개념을 **미리 던지고(복선)** 어디서 **본격으로 받는지(회수)** 정리.
> 회수 장치 점검·중복 조정의 지도. 작성 2026-05-30.

## 참조 표기 두 방식
- **1~32편**(數·力·波·統·物·生): "**개념의 자리에서** 다시 본다" — 번호 없이 개념명으로 지시.
- **33~53편**(計·思·美·望): "**N.이름에서**" — 번호+이름으로 명시.
- 두 방식이 섞여 있어, 다음 라운드에서 표기 통일 여부 결정 필요(개정원칙 B-1 참고).

## I. 책 전체의 골격 — 두 개의 큰 아치

### 1) 원형(序–終) 아치
- **0.序幕**(굴원 『천문』, 170개의 답 없는 물음, "마흔 해 뒤") → **54.終幕**("序幕에서 출발한 자리로 돌아왔다", 굴원 회수, 같은 청년의 마흔 해 뒤).
- 序는 *물음의 시작*, 終은 *물음이 답이 아니라 삶이라는 결론*. 닫힌 고리.

### 2) 받아들임 ↔ 물음 아치 (책의 척추)
- **1.공리**(유클리드의 5번째 공준 = 증명 없이 *받아들임*) → … → **53.인간**("받아들임에서 시작해 물음으로 끝난다 / 두 행위는 한 얼굴").
- 중간 기둥: **6.불완전성**(어떤 체계도 자기를 정당화 못함) → **41.이성**("모든 받아들임은 증명되지 않는다") → **44.침묵**(말할 수 없는 것).

## II. 주요 복선-회수 갈래(thread)

### ① 받아들임·자기지시·한계
`1.공리(복선: 공리는 선택)` → `3.극한과무한(칸토어 대각선)` → `6.불완전성(본격: 괴델, 자기지시)` → `34.계산(본격: 튜링 정지문제, 6과 동형)` → `44.침묵(종합: 괴델·튜링·비트겐슈타인 한 경계)` → `41.이성(받아들임 명제)` → `53.인간·54.終幕(회수)`
- 핵심 회수어: "어떤 체계도 자기를 완전히 정당화할 수 없다."

### ② 시간의 화살·엔트로피
`11.엔트로피(본격)` → `37.시간(통계적 화살로 시간 감각 설명)` / `29.세포(생명=음엔트로피)` / `33.정보(섀넌·볼츠만 엔트로피 동형)` / `48.비극(비가역성=비극)` / `52.공존(개방계의 국지적 질서)`
- 미리 던짐: 5.대칭의 "연필 세우기"(→19.깨짐), 9.동시성의 절대시간 붕괴(→37.시간).

### ③ 측정·실재 (波 단원 → 思 단원)
`13.양자` → `14.파동과확률` → `15.불확정성` → `16.경로` → `17.얽힘` → `18.측정(波의 수렴점)` → `38.실재(인식론적 위기)` → `39.인식(자기검증 불가)` / `43.경계(관찰자=대상)`
- 13~17이 모두 "측정의 자리에서 다시 본다"로 18에 수렴(건축적 의존). 18은 다시 38로 넘김.

### ④ 기하 → 중력 (數 → 力)
`1.공리(비유클리드, 가우스·로바쳅스키·보여이의 과학사 서사)` → `4.곡률(가우스·리만, 내재적 곡률·계량·측지선)` → `10.휨(본격: 일반상대성)` (+ `9.동시성`이 시공간 통합으로 다리)
- 1.공리의 가우스 침묵은 “먼저 본 사람이 말하지 않을 때 발견은 다시 고독하게 반복된다”는 과학사 모티프를 남긴다. 이 모티프는 4.곡률의 순수수학→물리학 회수, 6.불완전성의 토대 흔들림, 44.침묵의 말할 수 없음과 느슨하게 호응한다.
- 4.곡률의 "수학이 먼저 쓴 문장을 물리학이 찾아온다"는 축은 10.휨에서 일반상대성이론으로 회수한다. 단, 10에서는 가우스의 측량·지도 왜곡·비눗방울 예시는 반복하지 않고 중력=시공간 기하에 집중한다.

### ⑤ 대칭 → 깨짐 → 입자 → 우주 (統 단원 선형 축)
`5.대칭(군론·갈루아·뇌터 정리·연필 예고)` → `19.깨짐(자발적 대칭 깨짐·힉스)` → `20.입자(게이지 대칭·표준모형, 본격)` → `21.빅뱅` → `22.암흑` → `23.초끈` → `24.다중우주`
- 20.입자의 "표준모형 너머 다섯 빈자리"가 21·22·23으로 각각 회수. 23의 풍경 문제(10^500) → 24.

### ⑥ 물질 계층 (物 단원)
`25.원자(주기율표·원자번호)` → `26.결합(화학결합·자유에너지)` → `27.문턱(생명의 화학적 발생)` → `28.핵(핵력·핵에너지·윤리)`
- 25의 "원자번호=양성자 수"는 28.핵에서 물리적 근거로 회수. 26의 소수성 효과 → 27·29.

### ⑦ 생명 → 마음 (生 단원)
`29.세포(DNA·정보로서 생명)` → `30.진화(자연선택·뇌의 진화)` → `31.뇌(예측부호화·DMN·창발)` → `32.의식(어려운 문제)`
- 27.문턱의 RNA월드 → 29·30으로. 30의 "뇌가 자기를 응시" → 31. 31의 "패턴이 자기를 응시" → 32.

### ⑧ 자아 해체 (生 → 思)
`32.의식(자유는 해체의 자리에서)` → `42.해체(붓다 무아 + 데닛 서사중심)` → `43.경계(공·양자진공, 안과 밖)` (+ `39.인식`의 자아 불투명, `41.이성`의 자아 전제)
- 32가 명시적으로 40.의지·42.해체를 예고 → 회수.

### ⑨ 별의 잔해 · 우주의 자기인식
`21.빅뱅(원소 합성 시작)` → `25.원자` → `51.우주(본격: 별의 먼지·페르미 역설·우주가 자기를 앎)` → `52.공존(물질 순환)` → `53.인간(별 잔해가 의식이 됨)`
- ⚠ 같은 모티프 5회 등장 — 개정원칙 A-1 대상. 본격은 51, 나머지는 변주된 환기만.

### ⑩ 이름의 보존
`48.비극(본격: 자연은 누가 사라졌는지 안 적는다 / 헥토르·오이디푸스·햄릿·김길상)` → `53.인간(직접 재인용: 의미의 보존)`

### ⑪ 어긋남이 깊이를 만든다 (美 단원 미학 축)

### ⑭ 결정론 ↔ 자유

## IV. 점검 메모
- **수렴점 꼭지**(많은 복선이 모이는 곳): 18.측정, 44.침묵, 53.인간, 54.終幕. → 이 꼭지들은 회수 밀도가 높아 "포인터 모음"이 되기 쉬움. 개정원칙 B-2 우선 적용.
- **고아 복선 점검**: 던졌으나 회수가 약한 것 — 8.장의 "양자장으로 이어지는 복선"이 13.양자 이후에서 명시 회수되는지 확인 필요(현재 암시만).
- **순환 닫힘 확인**: 24.다중우주·53.인간·54.終幕이 모두 1.공리(유클리드 5공준)로 되돌아감 — 3중 회귀가 과하지 않은지 점검.
