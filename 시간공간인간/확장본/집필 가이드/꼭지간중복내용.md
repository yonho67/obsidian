# 《시간, 공간, 인간》 확장본 — 꼭지별 중복 내용 대장

> 각 꼭지에서 **다른 꼭지와 겹치는 항목**과 그 **본격(本格) 위치 / 회수 위치**.
> 표기: `항목 → N.꼭지(본격)` = 본격은 그쪽 / `항목 ↔ N.꼭지(상호)` = 양쪽 비슷한 깊이(=조정 대상) / `항목 ← N.꼭지` = 여기가 본격, 그쪽이 환기.
> ⚠ = 거의 같은 깊이로 중복되어 다음 라운드 삭감 1순위. 작성 2026-05-30.

- 비유클리드/평행선 → 4.곡률(전개), 24.다중우주(순환 인용)
- 칸트 물자체·직관 → 38.실재(본격)
- "공리는 선택" 복선 → 53.인간·54.終幕(받아들임으로 회수)
- 가우스·로바쳅스키·보여이의 비유클리드 발견 서사 ← 1.공리(본격). 4.곡률·6.불완전성·44.침묵에서는 이 서사를 다시 풀지 말고 “침묵/고립/받아들임” 모티프로만 짧게 환기.

### 2. 허실
- 허수 i·복소평면·오일러 공식 = 2.허실 본격. 14.파동과확률에서는 파동함수·확률진폭의 언어로 회수
- 슈뢰딩거 방정식은 2.허실에서 `i`의 물리학적 효용을 보여 주는 예고, 양자 해석 본격은 14.파동과확률
- 허수 시간·무경계 → 21.빅뱅(본격). 2.허실에서는 "시간을 공간처럼 다루게 하는 수학적 회전"까지만
- 복소수 회전 ↔ 5.대칭(상호언급). 5에서는 회전·대칭의 구조만 받고 허수 역사는 반복 금지

### 3. 극한과 무한
- ε-δ 극한·제논·소진법·칸토어 대각선 = 3.극한과무한 본격
- 연속체 가설의 독립성 → 6.불완전성에서 공리계와 독립 명제 본격. 3에서는 예고와 충격만
- 칸토어 대각선→튜링 변형 → 34.계산(본격). 3에서는 정지 문제를 한 문단 예고만
- 칸토어의 생애 서사는 3에서 충분히 전개. 6·34에서는 반복 금지

### 4. 곡률
- 내재적 곡률·계량·측지선 = 4.곡률 본격
- 리만 기하학의 물리학적 회수·일반상대성 → 10.휨(본격). 4에서는 장방정식의 의미까지만 예고하고, 수성 근일점·빛 휨·블랙홀·중력파는 10에서 전개
- 비유클리드 ← 1.공리. 가우스·로바쳅스키·보여이의 비극 서사는 1에서 이미 전개했으므로 4에서는 반복 금지
- 지도 왜곡·메르카토르 도법은 4.곡률의 곡률 예시. 49.미술·50.건축에서 세계를 그리는 방식으로 재등장할 경우, 지도 수치와 도법 설명은 반복하지 말 것
- 비눗방울·표면장력은 4에서 자연 형태 예시. 26.결합·50.건축에서 힘의 균형으로 회수 가능하되 플라토 법칙 설명은 짧게
- 회전군 SO(3) ↔ 5.대칭

### 5. 대칭
- 대칭·군·불변량·뇌터 정리 = 5.대칭 본격
- 갈루아의 5차방정식 서사와 뇌터의 차별/망명 서사 = 5.대칭 본격. 6.불완전성·20.입자에서는 반복 금지
- 자발적 대칭 깨짐 → 19.깨짐(본격). 5에서는 연필 예시와 “가능성/현실” 구조까지만
- 게이지 대칭·표준모형 → 20.입자(본격). 5에서는 U(1), SU(2), SU(3)와 국소 대칭의 직관만
- 복소수 회전 ← 2.허실. 5에서는 회전·변환 구조만 받고 허수 수학사는 반복 금지
- 공간 변환·회전 ← 4.곡률. 5에서는 곡률 설명 반복 금지

### 6. 불완전성
- 괴델 정리의 메아리 → 44.침묵(종합)
- 자기지시 구조 → 42.해체 / 34.계산(튜링)
